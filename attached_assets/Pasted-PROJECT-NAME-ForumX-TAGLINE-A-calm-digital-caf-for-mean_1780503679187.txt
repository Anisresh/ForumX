PROJECT NAME

ForumX

TAGLINE

A calm digital café for meaningful conversations.

====================================================
OVERVIEW
========

Build a complete production-quality social communication platform called ForumX.

ForumX should feel like:

* A calm digital café
* A warm online community
* A place to make friends
* A place to meet people
* A place for meaningful conversations

ForumX is NOT:

* A Discord clone
* A Slack clone
* A gaming platform
* A social media feed
* A server-based platform

Keep the experience simple, beautiful, warm, modern, and welcoming.

====================================================

==========

* Tailwind CSS

Backend:

* Firebase Authentication

AI:

* Gemini
* Genkit

IMPORTANT:

The entire application must function normally even if Gemini is unavailable.

* Real-time messaging
* Typing indicators
* Emoji support
* Message timestamps
* Online user count
* Reply to messages
* Edit messages (5 minutes)
* Delete own messages
